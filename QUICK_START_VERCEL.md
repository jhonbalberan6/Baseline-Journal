# Quick Start: Deploy to Vercel (5 Minutes)

## 1️⃣ Push to GitHub

```bash
git init
git add .
git commit -m "Initial: Baseline Journal"

# Then visit https://github.com/new
# Create repo "baseline-journal"
# Copy commands below and paste:

git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/baseline-journal.git
git push -u origin main
```

## 2️⃣ Deploy to Vercel

1. Go to https://vercel.com/new
2. Click "Import Project"
3. Paste: `https://github.com/YOUR_USERNAME/baseline-journal`
4. Click "Deploy"
5. Wait 2-3 min ⏳
6. Copy the URL that appears at top

## 3️⃣ Install on Phone

### iOS
1. Open Safari → Visit your Vercel URL
2. Tap Share → "Add to Home Screen"
3. Tap "Add" ✓

### Android
1. Open Chrome → Visit your Vercel URL
2. Tap ⋮ (menu) → "Install app"
3. Confirm ✓

---

**Done!** Your private journaling app is on your phone. All data stays on your phone. 🔒
